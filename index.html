Syllabus

Welcome to Advanced Data Science I & II. This course is designed to provide a comprehensive understanding of various advanced topics in data science. Here are some key details:

  • The class will be graded based on equally weighted projects.
  • Attendance is mandatory and represents 5% of the overall grade.
  • Class meets every Tuesday and Thursday from 8:30 AM to 9:50 AM.
  • All assignments should be submitted via GitHub Classroom. Links will be provided on CoursePlus.

Assignments

This syllabus outlines the key assignments and projects for Advanced Data Science I & II.

  • Assignment 1: Data Analysis - Introduction to data analysis, exploring datasets, and visualizing results.
  • Assignment 2: Text Analysis Project - Techniques in natural language processing and sentiment analysis.
  • Assignment 3: Image Analysis and Principal Component Analysis - Understanding image data, implementing PCA, and image classification.
  • Assignment 4: Data Cleaning and Databases - Techniques in data cleaning, handling missing data, and working with databases.
  • Assignment 5: Prediction and Machine Learning - Building predictive models and understanding different machine learning algorithms.
  • Assignment 6: Deep Learning and Neural Networks - Exploring deep learning frameworks and constructing neural networks.
  • Assignment 7: Study Design and Grant Writing - Learning about scientific study design and effective grant writing skills.
  • Assignment 8: Capstone Presentation - Presenting a comprehensive project that encapsulates learned skills and knowledge.
