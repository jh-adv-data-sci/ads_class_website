General Guidelines

References must be numeric [1] or author year style (Muschelli 2023).

Parenthetical statements should occur in the bracket or parentheses; [as seen in 1] or (as seen in Muschelli 2023).

There should be a space between the prior word and the citation.

No nested parentheses in parenthetical citations eg: (Muschelli (2023))

Tables and Figures

Tables should be in a fixed width font. Decimals should line up, and leading zeros should be added. Only meaningful digits should be given. Use table lines sparingly and with some thought.

Equations and Code

Equations should have punctuation at their end and only be numbered if referred back to. Code should be indented at 3 or more spaces.

Other Considerations

Consistent tense should be used throughout the document. Check your references for proper capitalization and consistent abbreviations.
